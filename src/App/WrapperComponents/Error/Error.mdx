# Error
Conjunto de componentes que são utilizados para informar ao usuário quando ocorre algum tipo de erro, sendo eles:

- **Not Found**: Página não encontrada.
- **Boundary**: Erro na renderização da página.
- **Undef**: Ocorreu algum erro.
