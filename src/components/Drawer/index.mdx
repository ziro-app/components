# Drawer
Este é um componente *Stand-alone*, mas, seu uso mais comum é dado pela composição do mesmo com outros dois componentes, sendo eles **DrawerPanel** e **Header**.

## DrawerPanel
Componente responsável por apresentar as opções que podem ser acessadas pelo menu de navegação **Drawer**, bem como mostrar informações referentes ao usuário.

### Properties [DrawerPanel]

<Props of={DrawerPanel} />

## Header
Componente de cabeçalho, responsável por apresentar alguma mensagem para o usuário e também por abrir/fechar o componente menu **Drawer**.
